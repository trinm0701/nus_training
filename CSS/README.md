# Bài giảng CSS căn bản

Thư mục này chứa bài giảng tiếng Việt cho buổi training CSS.

## Cách mở

Mở file `index.html` bằng trình duyệt:

```bash
open CSS/index.html
```

## Nội dung chính

- CSS Syntax: selector, property, value, declaration.
- ID & class: khi nào dùng `.class`, khi nào dùng `#id`.
- CSS How To: inline, internal, external.
- Cascading Order: importance, origin, specificity, source order.
- CSS Selectors: element, group selector, descendant, child, attribute selectors.
- CSS Properties: animation, background, border, color, cursor, display, flex, grid, font, size, position, spacing, overflow, opacity, line-height, list-style, float; có thêm playground bằng checkbox, select, slider và nút `Show code` sinh CSS theo lựa chọn hiện tại.
- CSS pseudo class/element: `:link`, `:visited`, `:hover`, `:active`, `:checked`, `:disabled`, `:empty`, `::before`, `::after`, `:first-child`, `:last-child`, `:nth-child()` và các biến thể liên quan.

## Gợi ý dùng trong buổi train

1. Đi theo thứ tự các mục trên thanh điều hướng.
2. Dừng ở phần selector để cho học viên tự đọc HTML và đoán selector nào sẽ match.
3. Ở phần CSS Properties, cho học viên tự bật/tắt property, chọn option, kéo slider, sau đó bấm `Show code` để đối chiếu CSS đang được apply.
4. Dùng phần mini lab cuối bài để học viên tự style một card sản phẩm.
